## [1.0.0] - 2025-07-25
### Added 
- Schemas
  - Primitives Schemas
    - Boolean Schema
    - Null Schema
    - Number Schema
    - String Schema
  - Array Schema
  - Batch Schema
  - Literal Schema
  - Object Schema
- Instance based Factory (Sanitizr.php)
- Static based Factory (SanitizrStatic.php)
- SafeParse Mode
- Value Object Trait for easier schema creation for a Value Object
- Constant Based Error Messages
- Unittest Framework (~PHPUnit)
